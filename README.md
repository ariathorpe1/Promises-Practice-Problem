# Fetch API Project Instructions

This project is designed to help you **practice making fetch() calls** to public APIs

You are provided with **HTML** and **CSS** — your job is to write the **JavaScript** needed to make everything work.

---

## Overview
Write a `script.js` file that:
1. Adds click listeners to each button.
2. Makes a `fetch()` request to the matching API.
3. Displays the data in the correct `<div>`,`<p>`, or `<img>` section.

---

## APIs to Use

### Dog CEO API
Get a random dog image  
`https://dog.ceo/api/breeds/image/random`  
- Display the image inside the `<img id="dog-img">`.

---

### BoredAPI
Get a random activity suggestion  
`https://www.boredapi.com/api/activity`  
- Display the `"activity"` text inside `<p id="bored-output">`.

---

### AdviceSlip API
Get random advice  
`https://api.adviceslip.com/advice`  
- Display the `"slip.advice"` inside `<blockquote id="advice-output">`.

## Requirements
- Use `.then()` **or** `async/await`.
- Don’t edit the HTML or CSS.
